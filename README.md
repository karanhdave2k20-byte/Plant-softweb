# SmartCode- Code Review

## Project Idea
SmartCode is an AI-based code review assistant that helps students and developers analyze and improve their code.

## Tech Stack
Frontend: React, HTML, CSS  
Backend: Node.js, Express  
Database: Supabase  
AI: OpenAI API (planned)

## Folder Structure

smartcode-frontend → React frontend application  
smartcode-backend → Node.js backend API  

Frontend contains:
- Components
- Pages
- Services

Backend contains:
- Controllers
- Routes
- Config
